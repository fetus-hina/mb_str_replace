mb_str_replace (line 27)

マルチバイト対応 str_replace()

  • return:

    内の $search を $replace で置き換えた文字列

    この関数の $search, $replace, $subject は配列に対応していますが、 $search, $replace が配列の場合の挙動が PHP 標準の str_replace() と異なります。

mixed mb_str_replace (mixed $search, mixed $replace, mixed $subject, [string $encoding = 'auto'])
  • mixed $search: 検索文字列（またはその配列）
  • mixed $replace: 置換文字列（またはその配列）
  • mixed $subject: 対象文字列（またはその配列）
  • string $encoding: 文字列のエンコーディング(省略: 内部エンコーディング)
